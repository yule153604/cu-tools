# 中国联通自动化脚本

这是一个专为中国联通 APP 设计的自动化脚本，采用 Python 异步 IO 编写，支持多账号并发运行。当前版本将业务逻辑收敛为单一 `Unicom` 类，统一处理登录、请求、日志与异常。

## 当前模块

脚本登录成功后，按以下顺序依次执行各活动任务（单个任务失败不影响其他任务）：

| 顺序 | 任务标签 | 说明 | 限制 |
|------|----------|------|------|
| 1 | 权益超市 | 优享权益领取、每日抽奖 | 全量账号 |
| 2 | 签到区 | 每日签到、月签有礼、任务中心、账户明细 | 全量账号 |
| 3 | 新疆联通 | 每月主题抽奖、每日打卡抽奖 | 仅新疆归属地 |
| 4 | 天天领现金 | 打卡与抽奖 | 全量账号 |
| 5 | 联通祝福 | 登录后按 Banner 执行任务与抽奖 | 全量账号 |
| 6 | 安全管家 | 签到、积分任务、助理代接、号段拦截 | 全量账号 |
| 7 | 通通乡村 | 农场登录、新手引导、签到、任务、种植充能收获 | 全量账号 |
| 8 | 云盘乘风活动 | 碎片任务、作品制作、芒果 TV 联动、抽奖 | 需云盘登录成功 |
| 9 | 上传大比拼 | 开启冲榜、上传文件、省份排名、抽奖 | 需云盘登录成功 |
| 10 | 商都福利 | 月充福利签到与抽奖 | 仅河南归属地 |

已移除模块：云手机、联通阅读、爱听专区。

## 功能说明

### 登录

- 使用抓包或 `login.py` 获取的 `token_online` 登录
- 登录成功后自动尝试云盘登录，为云盘乘风活动、上传大比拼提供 `userToken`
- 多账号使用 `@` 分隔，每个账号独立 `httpx.AsyncClient` 与会话

### 签到区

- 每日签到、月签有礼自动领取
- 任务中心自动执行任务并领取奖励（含 0329 繁忙退避）
- 查询最近兑换类账户明细

### 权益超市

- 优享权益每日领取（优先惊喜权益，支持人机验证重试）
- 多账号按序号错峰 0.15s，减轻 10:00 抢领并发压力
- 每日自动抽奖并输出中奖奖品

### 新疆专区

- 按当前月份自动生成活动 ID（如 `Jul2026Act`）
- 每月主题抽奖 + 每日打卡（daka）抽奖
- 非新疆归属地自动跳过

### 天天领现金 / 联通祝福

- 天天领现金：授权、打卡、周末特殊抽奖类型
- 联通祝福：按活动 Banner 动态执行任务与抽奖

### 安全管家

- 签到与积分任务领取
- 助理代接、号段拦截等月度任务

### 通通乡村

- 新手引导、每日签到、任务领取
- 跳转任务、垃圾清理任务
- 种植、充能、收获与复种循环

### 云盘乘风活动

- 碎片阶段推进、文件搬运、AI 任务
- 芒果 TV 模板作品提交
- 自动消耗抽奖次数

### 上传大比拼

- 进入活动页、开启省份冲榜
- 上传文件获取抽奖次数（支持自定义文件名与内容）
- 省份排名查询与抽奖

### 商都福利

- 河南归属地校验
- 月充福利签到与抽奖（支持节日奖池映射）
- 响应 SM2 解密（需安装 `gmssl`）

## 快速开始

### 1. 安装依赖

```bash
pip install httpx pycryptodome gmssl
```

> `gmssl` 仅商都福利需要；不跑商都任务时可不装。

### 2. 配置账号

脚本通过环境变量 `chinaUnicomCookie` 读取 `token_online`。

#### 获取 token_online

**方式一：使用登录脚本（推荐）**

1. 编辑 `login.py`，填入手机号和短信验证码
2. 运行 `python login.py`
3. 控制台输出的字符串即为 `token_online`

```python
Mobile = "18812345678"  # 填入手机号
SMS_code = "123456"     # 填入短信验证码
```

**方式二：抓包获取**

1. 使用抓包工具抓取中国联通 APP 请求
2. 找到 `https://m.client.10010.com/mobileService/onLine.htm`
3. 提取请求体中的 `token_online`

#### 配置环境变量

```bash
# 单账号
export chinaUnicomCookie="your_token_online_string"

# 多账号（@ 分隔）
export chinaUnicomCookie="token1@token2@token3"
```

### 3. 可选环境变量

| 变量 | 默认值 | 说明 |
|------|--------|------|
| `UNICOM_TTXC_GARBAGE_WAIT` | `28` | 通通乡村垃圾任务等待秒数 |
| `UNICOM_TTXC_GROW_MAX_CHARGE_PER_LAND` | `20` | 单块地最大充能次数 |
| `UNICOM_TTXC_HARVEST_WAIT` | `3` | 收获等待秒数 |
| `UNICOM_YPHD_MGTV_IMG_FID` | 空 | 云盘乘风芒果 TV 模板图片 fid |
| `UNICOM_CLOUD_BATTLE_FILE` | `文本.txt` | 上传大比拼文件名 |
| `UNICOM_CLOUD_BATTLE_CONTENT` | `1` | 上传大比拼文件内容 |
| `SHANGDU_LOTTERY_MAX` | 剩余次数 | 商都福利单次最多抽奖次数 |

### 4. 运行脚本

```bash
python 中国联通.py
```

### 5. 日志说明

控制台输出格式：`[HH:MM:SS] [账号N] [任务标签] 消息`

- 登录成功显示脱敏手机号，如 `138****5678`
- 各任务以标签区分：签到区、权益超市、通通乡村、云盘乘风活动等
- 抽奖、领取结果会输出具体奖品或接口摘要

## 注意事项

- **HTTP/2**：默认关闭，使用 HTTP/1.1 提升稳定性
- **任务隔离**：`run()` 内各任务独立 try/except，单任务异常不阻断同账号其他任务
- **归属地限制**：新疆专区、商都福利会按登录返回的省份自动跳过非目标地区
- **云盘依赖**：云盘乘风活动、上传大比拼依赖 `cloudpan_login` 成功；失败时这两项目自动跳过
- **隐私安全**：手机号等敏感信息日志脱敏；请勿将 `token_online` 提交到公开仓库
